# PROYECTO MÓDULO 2 — RESUMEN DE EJERCICIO 1 Y 2

## EJERCICIO 1 — API, PYTHON Y MYSQL

En este ejercicio se trabaja con una API de películas.

Los pasos principales son:

- Extracción de datos desde la API con requests.

- Limpieza con pandas (convertir booleanos, listas de subtítulos y valores NaN).

- Creación de la base de datos “films_ejercicio_2” y de la tabla “films”.

- Inserción de las 100 películas usando Python y mysql-connector-python.

- Realización de consultas SQL como:

-  Películas > 120 minutos

-  Películas con subtítulos en español

-  Películas adultas

-  Película más antigua

-  Promedio de duración por género

-  Películas por año y año con más películas

-  Búsquedas por título con LIKE

## EJERCICIO 2 — CONSULTAS SQL EN “SAKILA”

Se realizan consultas directamente sobre la base de datos “sakila”, como:

- Títulos sin duplicados

- Películas con clasificación PG-13

- Películas con “amazing” en la descripción

- Películas de más de 120 minutos

- Listado de actores

- Actores cuyo apellido contiene “Gibson”

- Películas según clasificación o año

- Total de películas alquiladas por cliente y por categoría

- Actores que participan en una película específica

- Películas entre 2005 y 2010

Ambos ejercicios sirven para practicar extracción de datos, limpieza, SQL básico y consultas sobre bases de datos reales.
